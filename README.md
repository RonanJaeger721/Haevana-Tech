# Haevena Tech Website

Premium static website concept for Haevena Tech, covering electric gates, CCTV, access control, alarms, electric fencing, video intercoms, and domestic solar backup.

## Preview

Open `index.html` in a browser. The site is static and does not require a build step.

## Project Structure

- `index.html` - page content and sections
- `styles.css` - responsive layout, visual design, animation styling
- `script.js` - hero scan, service selector, blueprint, and walkthrough interactions
- `assets/` - cropped installation and product imagery
